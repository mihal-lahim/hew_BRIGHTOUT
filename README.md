# DX21 ゲームプロジェクト

## 📖 プロジェクト概要

DirectX 11 を使用した3Dゲームプロトタイプです。プレイヤーが「人間」と「電気」状態に変身して、マップを探索するゲームです。

### 主な機能

- **プレイヤーコントローラー**: B ボタンで人間 ↔ 電気状態に変身
- **重力システム**: リアルな物理演算による落下
- **AABB 衝突判定**: 正確なオブジェクト間の当たり判定
- **カメラシステム**: トップダウンカメラの実装
- **マップシステム**: 4つのタイル式地面
- **オブジェクト管理**: 電柱や建物などの静的オブジェクト管理

---

## 🎮 ゲームコントロール

| 入力 | 機能 |
|------|------|
| **左スティック** | プレイヤーの移動 |
| **右スティック** | カメラの回転 |
| **A ボタン** | ダッシュ（電気状態のみ） |
| **B ボタン** | 状態切り替え（人間 ↔ 電気） |
| **X ボタン** | カメラ切り替え |
| **Right Shoulder** | ダメージ（デバッグ用） |
| **Tab キー** | デバッグカメラ ON/OFF |
