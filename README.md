# DX21 ゲームプロジェクト

## 📖 プロジェクト概要

DirectX 11 を使用した3Dゲームプロトタイプです。プレイヤーが「人間」と「電気」状態に変身して、マップを探索するゲームです。

### 主な機能

- **プレイヤーコントローラー**: B ボタンで人間 ↔ 電気状態に変身
- **重力システム**: リアルな物理演算による落下
- **AABB 衝突判定**: 正確なオブジェクト間の当たり判定
- **カメラシステム**: トップダウンカメラの実装
- **マップシステム**: 4つのタイル式地面
- **オブジェクト管理**: 電柱や建物などの静的オブジェクト管理

---

## 🎮 ゲームコントロール

| 入力 | 機能 |
|------|------|
| **左スティック** | プレイヤーの移動 |
| **右スティック** | カメラの回転 |
| **A ボタン** | ダッシュ（電気状態のみ） |
| **B ボタン** | 状態切り替え（人間 ↔ 電気） |
| **X ボタン** | カメラ切り替え |
| **Right Shoulder** | ダメージ（デバッグ用） |
| **Tab キー** | デバッグカメラ ON/OFF |

## 🛠️ ビルド方法

### 必須環境
- **Visual Studio 2022** 以上
- **Windows 10/11**
- **DirectX 11** SDK
- **C++ 14** 以上

### ビルド手順

1. Visual Studio 2022 で `.sln` ファイルを開く

2. **ソリューション構成** を選択
- Debug (デバッグ版)
- Release (最適化版)

3. **ビルド** → **ソリューションのビルド** をクリック

4. **デバッグ** → **デバッグの開始** (F5) で実行

## 🏗️ クラス構成

### Player クラス
プレイヤーキャラクターを管理します。
- **状態管理**: 人間 / 電気の2つの状態
- **移動**: WASD またはゲームパッド入力
- **重力**: Y軸方向の落下
- **衝突判定**: AABB ベースの当たり判定

### GameObject クラス
すべての静的オブジェクト（電柱、建物など）の基底クラス。


### ObjectManager クラス
すべてのゲームオブジェクトを一元管理します。

### Map クラス
地面（タイル）を管理・描画します。
